# 知乎用户 - 24小时动态可视化

这个脚本的初衷是分析知乎用户的活动轨迹，以一天24小时为一周期。可以判别用户所处的时区，比如自称在美国生活的用户，动态轨迹应该集中于北京时间凌晨至清晨那段时间。

## 实现方法

- 读取用户动态接口数据
- 解析json数据
- 将动态轨迹收录于一个大小为24的List，对应一天二十四小时
- 可视化List
