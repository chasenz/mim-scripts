# 布隆过滤器

布隆过滤器可以有效解决大集合中重复元素的判断，同时运用场景也多样，比如单词拼写错误判断，垃圾邮件过滤，URL去重。

它的优点是空间效率和查询时间都比一般的算法要好的多，缺点是有一定的误识别率和删除困难，称之为假阳性（false positive）。比如网站在注册页面判断用户名是否已经被注册，可能会出现未被注册的名称显示已注册。但布隆过滤器不会出现假阴性（false negative），也即已被注册的用户名显示未被注册的情况。

## 算法

1. 初始化$m$位的数组，并将值设定为$0$;
2. 设置$k$个哈希方法，$h_1(x), h_2(x), ... , h_k(x)$，哈希方法的下标数字即为种子（seed）;
3. 计算$h_i(x) \% m$，并将该下标的数组设为$1$，重复$k$次；
 
>根据预计处理数据量的个数$n$和允许的假阳性概率$p$，可以计算出最优的数组大小$m$和哈希方法个数$k$
>
> $$ m = - \frac{n ln(p)}{ln2^2} $$(1)
> $$ k = \frac{m}{n}ln2$$(2)

根据上述公示（1）和（2）可以计算出最优的数量
